# SJA TRANSPORT Frontend

A modern and responsive logistics and transport website built with React and Vite.

## Features

- Responsive UI
- Shipment Booking
- Gallery
- Services
- Contact Page
- Feedback
- Admin Login Integration

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Axios

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
